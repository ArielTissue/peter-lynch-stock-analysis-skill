# Peter Lynch Stock Analysis Skill

A Codex skill for analyzing individual stocks with a Peter Lynch-inspired stock selection framework.

## Install

```bash
python ~/.codex/skills/.system/skill-installer/scripts/install-skill-from-github.py \
  --url https://github.com/ArielTissue/peter-lynch-stock-analysis-skill/tree/main/skills/peter-lynch-stock-analysis
```

After installing, restart Codex.

## Example Prompts

```text
Use Peter Lynch stock analysis to analyze COST.
```

```text
用彼得林奇框架分析一下 DORM 是否适合加入观察清单。
```
